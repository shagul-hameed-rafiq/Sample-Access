# 🏥 MedLab AInsights

An AI-powered full-stack medical lab report analysis system designed to assist technicians and patients in understanding lab results quickly and accurately.

---

## 🚀 Project Overview

MedLab AInsights is a complete web application that allows users to:

* Enter patient details and lab test values
* Automatically detect abnormal values (LOW / NORMAL / HIGH)
* Generate structured medical reports
* Provide meaningful insights from lab data

The system reduces manual interpretation effort and improves efficiency in clinical workflows.

---

## 🧠 Key Features

* ✅ Rule-based abnormality detection
* ✅ Full-stack architecture (Angular + .NET)
* ✅ Clean and structured UI for data entry
* ✅ Real-time validation of medical values
* ✅ Scalable backend for future AI/ML integration
* ✅ Modular design for easy extension

## ⚙️ Setup Instructions

### 1. Clone the Repository

```
git clone https://github.com/shagul-hameed-rafiq/Medlab-First-Site.git
cd Medlab-First-Site
```

---

### 2. Run Frontend

```
cd Frontend
npm install
ng serve
```

Frontend runs on:
👉 http://localhost:4200

---

### 3. Run Backend

```
cd Backend
dotnet restore
dotnet run
```

Backend runs on:
👉 http://localhost:5000 (or configured port)
